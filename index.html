How to Use This Application: A Step-by-Step Guide

Follow these steps to effectively use the application and make the most of its quiz functionality:

  1. Import Queries from JSON File
    Before starting, ensure you have a set of settlement queries ready. Click the Settings button on the home page and choose Import Queries to upload the provided .json file located in your project folder. This will load the necessary quiz content into the application.
  2. Settings
    Within the Settings Page, you can Add or Manage your quiz queries:
    • Add New Queries: Create new settlement queries for future quizzes.
    • Edit Existing Queries: Update details of any saved queries.
    • Remove Queries: Delete queries that are no longer relevant.
    • Import Queries: Upload a JSON file to load a set of queries into the application.
    • Export Queries: Save your current set of queries as a JSON file for backup or sharing.
  3. Start the Quiz
    Return to the Home Page and click the Start Quiz button. Youâ€™ll be prompted to select a difficulty level, which controls the quiz timing and pace.
  4. Answer the Queries
    One query will be presented at a time: Read each query carefully. Submit your answer before the timer runs out. Receive instant feedback after submitting each response.
  5. Review Your Results
    Once all queries have been answered, a detailed results summary will be displayed. Review your answers alongside the correct ones. Identify which queries were answered correctly or incorrectly. Reflect on your performance to improve next time.
  6. Export Your Results
    For record-keeping or reporting: Use the Export All Answers to Excel feature to download an XLSX file containing your quiz results.
